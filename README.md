# DevPath AI — Templates

**DevPath AI** 프로젝트에서 공용으로 사용하는 템플릿 모음 레포지토리입니다.

## 🎯 목적

문서·코드·설정 템플릿을 한곳에서 관리하여 레포 간 일관성을 유지합니다.

## 📂 구성 (예정)

| 분류 | 내용 |
|------|------|
| `docs/` | 문서 템플릿 — 테스트 보고서, 회의록, 설계 문서 양식 |
| `github/` | Issue / PR 템플릿, 워크플로 템플릿 |
| `config/` | 환경 설정 템플릿 — Docker Compose, OAuth2 설정 등 |
