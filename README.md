# DevPath AI — Templates

**DevPath AI** 프로젝트에서 공용으로 사용하는 템플릿 모음 레포지토리입니다.

## 🎯 목적

문서·코드·설정 템플릿을 한곳에서 관리하여 레포 간 일관성을 유지합니다.

## 📂 현재 상태

이 레포는 현재 `CLAUDE.md`·`README.md` 2개 파일만 존재하는 **빈 셸**이다. 아래는 예정된 구성이며 아직 생성되지 않았다.

| 분류 | 내용 | 상태 |
|------|------|------|
| `docs/` | 문서 템플릿 — 테스트 보고서, 회의록, 설계 문서 양식 | 예정(TARGET) |
| `github/` | Issue / PR 템플릿, 워크플로 템플릿 | 예정(TARGET) |
| `config/` | 환경 설정 템플릿 — Docker Compose, OAuth2 설정 등 | 예정(TARGET) |
